# bootcamp_javascript_m2.tarea4
Tarea 4 - Modulo 2 - Diseño responsivo
Ejercicio Práctico - Diseño responsivo y Navbar con Bootstrap
Actividad Práctica: Diseño responsivo y Navbar con Bootstrap
Objetivo:
Aprender a transformar un proyecto web en una versión responsiva bajo el enfoque Mobile First, aplicando Flexbox, CSS Grid y Media Queries, probar la visualización en distintos dispositivos mediante el inspector de elementos, e integrar Bootstrap para añadir una barra de navegación.

Descripción de la tarea:
Continúa con el proyecto de la Lección 3 y realiza los siguientes pasos:

Diseño responsivo con Mobile First

Mantén tus estilos base pensados en pantallas pequeñas (móviles).

Aplica Flexbox para organizar elementos (ej.: cabecera, tarjetas de productos).

Utiliza CSS Grid para estructurar secciones (ej.: contenido + barra lateral).

Define al menos dos media queries para ajustar la visualización en tablet y escritorio.

Sugerencia: revisa que la página no tenga scroll horizontal innecesario.

Pruebas en dispositivos simulados

Abre tu proyecto index.html en el navegador.

Usa el inspector de elementos (herramientas de desarrollador) y activa la vista de dispositivos.

Prueba tu sitio en al menos tres tamaños de pantalla diferentes (ejemplo: móvil, tablet y desktop).

Haz una captura de pantalla de una de estas vistas y guárdala en la carpeta ASSETS/IMG/.

Integrar Bootstrap y Navbar

Agrega Bootstrap por CDN (revisa la documentación oficial para copiar el enlace correcto).

Inserta una Navbar de Bootstrap y modifica sus títulos (ej.: Inicio, Productos, Contacto).

Verifica que la barra sea colapsable en móvil (menú hamburguesa) y visible completa en escritorio.
